# LunarProject521
This is a multi layer Neural Network project being done for EGEC-529 at CSUF related to using OpenAI Gymnasium

Requires Python 3.13.12

Runs in a virtual enviroment that is activated by Scripts\Activate while in the folder directory.

install dependancies with pip install Requirements.txt.

To train a new model, call 'Agent.py' with a python command.

Total Rewards and Epsilon are saved for all episodes in data.txt
